# 摸鱼看书 · Win11 桌面端（Electron）

一条细长文字混在屏幕里，看起来像状态栏 / 日志 / 代码注释。进度与手机网页端走**同一个阿里云同步服务、同一个同步码**，双向自动同步。

## 运行

```bash
cd desktop
npm install      # 首次，下载 Electron
npm start
```

首次启动会从同步服务下载《覆汉》并缓存到本地（之后离线也能读、秒开）。

## 打包成免安装 exe（可选）

```bash
npm run dist     # 产物在 desktop/dist/thiefbook-desktop-1.0.0.exe
```

双击即用，绿色免安装，最贴合摸鱼场景。

## 用法

| 操作 | 键 / 方式 |
| --- | --- |
| **老板键**（秒隐/秒现） | `Alt + P` |
| 翻页 | `PageUp` / `PageDown`（也可鼠标滚轮）；翻到章末自动进下一章 |
| **快速选章节** | `Alt + G`，或右键菜单；输入关键词/章号，↑↓ 选，回车跳 |
| 打开设置 / 目录 | 在文字条上**右键** → 设置 / 目录 |
| 移动位置 | 按住文字条拖动（右键可「锁定位置」防误拖） |
| 退出 | 右键 → 退出 |

## 首次配置同步

右键 → 设置 → 填 **同步码**（和手机网页端一模一样的那串，如 `eric-2026`）。填好后阅读器会自动拉取云端进度并保持同步。

## 隐蔽特性

- 无边框、无标题栏、无最大化/最小化/关闭按钮
- 不在任务栏显示图标（`skipTaskbar`）
- 始终置顶，只显示背景色 + 文字
- 背景色/文字色/字号/行数/宽度/透明度/字体全部可调（默认深灰底 + 灰字，像代码注释）

## 与手机端同步的原理

- 书源：`${serverUrl}/books/fuhan.json`，与手机端同一份
- 进度接口：`GET/POST /api/progress`，按 `code + book` 存储，`last-write-wins`
- 进度模型：章节序号 + 章内百分比；桌面端把「页」映射成章内百分比，与手机滚动位置对齐
- 桌面端每翻页防抖上报；隐藏(Alt+P)时立即上报；恢复显示 / 每 20 秒拉取一次云端，捕捉手机端的最新进度
